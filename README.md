# Programming Lessons Telegram Bot

## Project Description

This project is an educational Telegram bot developed in Python. The bot helps beginners learn programming through structured lessons and an interactive interface.

The bot currently supports five programming languages:

- Python
- C++
- Java
- JavaScript
- C#

Users can select a programming language, browse available lessons, and navigate between lessons using built-in buttons. The project is designed to make learning programming more accessible and organized.

## Installation and Setup

To run this project locally:


### 1. Clone the repository

```bash

```
